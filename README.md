# PERFERENCE OF TOPOLOGIES

# INTRODUCTION

•	Network topology refers to the physical or logical arrangement of devices in a computer network. The preference for a topology depends on factors such as cost, reliability, scalability, performance, and ease of maintenance.

•	The preference of a network topology depends on factors like cost, reliability, scalability, speed, maintenance, and network size.

•	Mesh Topology

# 1. Point-to-point Topologies
A Point-to-Point topology is a network configuration in which two devices are directly connected through a dedicated communication link.
## Real-Time Scenarios
1. Leased Line Internet Connection

A company directly connects its branch office to headquarters using a leased line.

2. Bluetooth File Transfer

Two mobile phones connected directly through Bluetooth.

3. Satellite Communication

Communication between a satellite and ground station.

4. Point-to-Point Wireless Bridge

Two buildings connected using wireless antennas.

# 2.	Bus Topology
<img width="305" height="228" alt="image" src="https://github.com/user-attachments/assets/c5a411fb-d278-4c74-ac56-7ac942a79a9b" />

  

## Features
•	All devices are connected to a single communication cable called the backbone.

•	Data travels in both directions.

## Preference / Suitable Scenarios
•	Small networks 

•	Temporary networks 

•	Low-cost installations

## Advantages
•	Simple and inexpensive

•	Requires less cable 

•	Easy to install 

## Limitations
•	Backbone failure affects entire network 

•	Difficult fault detection 

•	Performance decreases with more devices 

## Real-Time Example
•	Early Ethernet LANs in small offices.

## Part E — Short Answer Questions
## Q1. Define Network Topology.
**Answer:**

Network topology refers to the physical or logical arrangement of computers and communication devices in a network.

## Q2. Why is Star Topology most preferred?
**Answer:**

Star topology is most preferred because it is easy to install, maintain, and troubleshoot. Failure of one node does not affect the entire network.

## Q3. What is the main advantage of Mesh Topology?
**Answer:**

The main advantage of mesh topology is high reliability and fault tolerance because multiple communication paths exist between devices.

## Q4. Why is Bus Topology considered low cost?
**Answer:**

Bus topology requires less cable and does not need a central device like a switch or hub, making it inexpensive.

## Q5. Explain one real-time application of Tree Topology.
**Answer:**

Tree topology is used in university campus networks where multiple departments are connected hierarchically for easy expansion and management.
